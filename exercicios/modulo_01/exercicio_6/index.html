Principais formatações

Tags semânticas

Importantes para a acessibilidade e SEO, pois ajudam os mecanismos de busca e leitores de tela a entenderem o significado do conteúdo.

Destaque <strong>

Utilizado para marcar palavras ou frases que possuem grande importância no contexto.
Indica prioridade ou alerta, e é interpretado como algo digno de atenção.

Ênfase <em>

Aponta uma ênfase no conteúdo, como se fosse uma entonação na fala.
Indica que o conteúdo é importante, mas não necessariamente crítico.

Marcador de texto <mark>

Marca um trecho de texto para destacar informações relevantes ou que precisam de atenção.
É útil para indicar partes importantes em um documento ou conteúdo.

Texto pequeno <small>

Aplica um estilo de texto menor.
Usado para notas de rodapé, avisos ou informações secundárias.

Texto excluído <del>

Indica que um trecho foi removido do conteúdo original, geralmente riscando o texto (linha sobreposta).
É útil em revisões, edições de documentos e históricos de alterações.

Texto inserido <ins>

Indica que um trecho foi inserido, visualmente sublinhando o texto.
Assim como o <del>, é usada em sistemas de controle de versão ou revisões de texto.

Tags não semânticas

Não alteram o significado do conteúdo, mas aplicam estilos visuais.
Essas tags são mais focadas na apresentação do que no significado.

Negrito <b>

Aplica apenas o estilo visual de negrito.

Itálico <i>

Aplica apenas o estilo visual de itálico.

Sublinhado <u>

Aplica o estilo visual sublinhado.

Texto sobrescrito <sup>

Aplica o estilo sobrescrito, geralmente usado para indicar expoentes, notas de rodapé ou referências.

Texto subscrito <sub>

Aplica o estilo subscrito, geralmente usado para fórmulas químicas, notações matemáticas ou referências.
