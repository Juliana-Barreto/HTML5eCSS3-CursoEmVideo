Testando cargas e imagens

Estou praticando a inserção de imagens utilizando HTML.
Através da tag <img>, é possível adicionar conteúdo visual às páginas, melhorando a apresentação e a compreensão das informações.
As imagens utilizadas neste projeto servem como exemplo para testar atributos como src, alt, width e height.
