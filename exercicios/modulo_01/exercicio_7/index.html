Outras formatações

Código-fonte <code>

Utilizado para exibir trechos de código ou comandos.

#include
int main() {
printf("Hello, World!");
return 0;
}

Pré-formatação <pre>

Usada para exibir texto pré-formatado, mantendo espaços e quebras de linha. 

    
      #include 
      int main() {
        int num;
        printf("Insira um número: ");
        scanf("%d", &num);
        printf("Você digitou: %d\n", num);
        return 0;
      }

Citações

Citações simples <q>

Usada para destacar citações curtas ou trechos de texto.

De acordo com Carl Sagan, Nós somos feitos de poeira das estrelas.

Citações completas <blockquote>

Usada para destacar citações longas ou trechos de texto.

Segundo Carl Sagan:

"Em algum lugar, algo incrível está esperando para ser conhecido. A ciência não é apenas compatível com a espiritualidade; é uma profunda fonte de espiritualidade. Quando reconhecemos nosso lugar em uma imensidão de eras-luz e no passar do tempo, quando compreendemos a complexidade, beleza e sutileza da vida, então esse sentimento, esse arrebatamento, combinado com a humildade e a alegria, é certamente espiritual."

Abreviações <abbr>

Usada para indicar abreviações ou siglas.

O HTML é uma linguagem de marcação usada para criar páginas web.

O CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML.
